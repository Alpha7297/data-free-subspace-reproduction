# 实验设计

初始条件为平铺的20x20弹簧质点网格,四个角点固定,之后对中心质点加入z方向外力,观察100帧之后的稳定态和平衡态。

以隐式欧拉100帧末态为基准,统计data_free结果的位置误差。

## 系统设置

| 参数 | 数值 |
|---|---:|
| 网格尺寸 | 20x20 |
| 质点数量 | 400 |
| 弹簧刚度 | 20.0 |
| 弹簧原长 | 1.0 |
| 固定点 | 四个角点 |
| 外力 | 中心质点z方向外力5.0 |
| 输出帧数 | 100 |
| 输出步长 | 0.01 |

误差为绝对误差,使用坐标单位。

# 结果分析

## 训练周期

观察到，在这个物理体系中

indim=20时，周期100k以上继续训练误差已经没有显著减低，甚至300k时误差相比100k更大的现象

indim=40时，周期200k以上继续训练误差已经没有显著减低，300k出现了误差相比100k更大的现象

因此可以认为，100k量级的训练周期对于本实验已经足够

## 误差分析

观察发现，静力学相对误差相比动力学有了显著下降，这说明网络学到了在低能态下的坐标投影关系

## 理论不足

实验结果中，误差最大值在受力点，并且受力点处有接近100%的巨大偏差

另外我尝试将布料放在重力场下，结果仿真结果行为严重非物理，雅各比伪逆数值不稳定

以上两点共同说明，如果没有在训练中就将外力写进势能，实际仿真中加上新的外力，在投影到低维空间时信息会严重损失。

该方法更适合学习低能态附近的稳定形变空间,不适合作为任意外力条件下的通用动力学模拟器。
