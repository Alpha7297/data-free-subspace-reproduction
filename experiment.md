# 实验设计

一维链条，给定相同初始条件，使用显式欧拉、隐式欧拉与 data_free neural subspace 分别进行仿真，并以显式欧拉结果为基准统计误差。

## 系统设置

| 参数 | 数值 |
|---|---:|
| 质点数量 | 100 |
| 弹簧刚度 | 2.0 |
| 弹簧原长 | 1.0 |
| 固定点 | 第 0 个质点 |
| 初始位置 | 沿 x 轴均匀排列 |
| 初始速度 | 右侧一半质点 y 方向速度为 -1 |
| 输出帧数 | 500 |
| 输出步长 | 0.01 |

误差为绝对误差。位置误差使用坐标单位，速度误差使用速度单位。

## 分析

1. 传统仿真方法具有数值一致性

隐式欧拉相对1000子步的显式欧拉误差极小，说明在这个物理系统中，传统积分方法结果基本一致，显式欧拉解可以作为合理的参考解。

2. data_free误差与训练周期并不直接相关

indim=20时，在训练周期5000到100000之间出现了诡异的非物理现象，而indim=10时并没有出现。

原因是data_free只是寻找平衡态附近的合理稳定解，而不是训练一个运动学轨迹，训练Loss虽然单调下降，但是仿真误差并不会

3. 相比于传统仿真方法，data_free这种方法在物理精度上仍不足
