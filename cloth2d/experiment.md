# 实验设计

初始条件为平铺的20x20弹簧质点网格,四个角点固定,之后对中心质点加入z方向外力,观察静力学稳定态和平衡态。

以传统迭代法为基准,统计data_free结果的位置误差。

## 系统设置

| 参数 | 数值 |
|---|---:|
| 网格尺寸 | 20x20 |
| 质点数量 | 400 |
| 弹簧刚度 | 20.0 |
| 弹簧原长 | 1.0 |
| 固定点 | 四个角点 |
| 外力 | 中心质点z方向外力3.0 |
| 输出步长 | 0.01 |

误差为绝对误差,使用坐标单位。

# 结果分析

## 训练周期

观察到，在这个物理体系中

周期100k左右已经足够，更大的训练周期对误差没有显著改善

动力学实验中，400k迭代后雅各比伪逆出现了严重退化，说明更多的迭代次数可能已经过拟合

## indim分析

indim=20结果显著优于indim=40

indim=40时严重病态


## 理论不足

静力学实验仍然相对传统方法误差巨大，这说明如果没有在训练中就将外力写进势能，实际仿真中加上新的外力，在投影到低维空间时信息会严重损失。

并且雅各比伪逆数值严重不稳定，经常出现不可逆现象。

该方法不适合作为外力条件下的静力学求解器。
